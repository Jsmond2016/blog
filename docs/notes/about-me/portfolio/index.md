# 个人作品集

这里不是简单的项目列表，而是我过去几年持续沉淀的内容资产总览。整体可以分成三类：用教程沉淀方法，用项目验证能力，用独立站点承接长期学习。

## 先看什么

- 如果你想快速了解我的输出风格，优先看[教程系列](/notes/about-me/portfolio/tutorials)。
- 如果你更关心我做过什么实际工具，优先看[重点项目](/notes/about-me/portfolio/projects)。
- 如果你想了解我的学习广度和持续性，可以看[其他站点笔记](/notes/about-me/portfolio/other-sites)。

## 内容结构

### 教程系列

这部分体现的是“把经验整理成体系”的能力。内容并不只停留在知识点罗列，而是尽量按学习路径、场景和实践顺序组织。

- 适合想快速了解我内容表达能力和技术拆解方式的读者；
- 重点体现前端延展到 Python、Chrome 扩展、Electron 等方向的学习与实战输出。

### 重点项目

这部分体现的是“针对真实问题做工具”的能力。选出的项目都偏向工作流优化、开发提效和个人使用场景，不是为了凑数量。

- 适合想了解我工程实践和产品意识的读者；
- 重点展示浏览器插件、编辑器插件、桌面工具和数据看板类项目。

### 其他站点笔记

这部分体现的是“长期学习积累”的稳定性。它们不一定都属于主线作品，但能反映出我持续输入、拆分主题并长期维护内容的习惯。

- 适合想了解我知识覆盖范围和学习节奏的读者；
- 重点展示算法、Go、Java 和日常学习记录等独立站点。
