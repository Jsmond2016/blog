# 前端进阶

这一栏主要放从“会写页面”走向“理解框架和工程实践”的内容。相比基础知识，这里的文章更偏向框架理解、场景拆解和项目经验。

## 关注重点

- Vue、React 及周边生态；
- 前端路由、组件通信等框架核心问题；
- TypeScript 进阶使用；
- 工程化与项目组织方式；
- 真实项目中更容易遇到的实践问题。

## 适合什么时候看

- 当你已经具备基础开发能力，开始接触框架和工程项目时；
- 当你不满足于“会用”，想进一步理解设计思路和常见问题时；
- 当你准备把零散经验整理成更稳定的方法论时。

## 阅读建议

- 如果你正处在框架学习阶段，可以先看 Vue、路由、组件通信相关主题；
- 如果你更关注项目落地，可以结合工程化和实战类内容一起看；
- 这一栏适合带着问题回看，而不是一次性顺序刷完。
