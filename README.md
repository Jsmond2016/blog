# vitepress @sugarat/theme
简约风的 [Vitepress 博客主题](https://theme.sugarat.top)

## Usage
先安装 `pnpm`

```sh
npm i -g pnpm
```

安装依赖
```sh
pnpm install
```

开发启动
```sh
pnpm dev
```

构建
```sh
pnpm build
```

预览产物
```sh
pnpm serve
```
