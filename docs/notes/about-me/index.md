# 关于我

我是黄同学，一名前端工程师，也是一名喜欢把学习过程沉淀成内容资产的长期主义学习者。

我现在这套博客，不只是为了放笔记，而是希望把自己的技术成长路径整理清楚。主线集中在前端基础、工程实践、浏览器扩展、Electron 实战，以及围绕真实问题做出来的小工具和教程站点。

## 我在持续做的事情

- 把零散知识整理成有结构的教程和笔记；
- 把工作和个人场景里的重复问题做成工具；
- 把日常学习拆到不同主题站点里长期维护；
- 持续补齐前端之外的语言和工程能力。
