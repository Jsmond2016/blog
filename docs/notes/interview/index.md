# 面试和八股文

这里主要整理前端面试准备过程中常见的问题、知识点复盘和阶段性总结。它既包括常见八股，也会保留我自己的理解和整理方式。

## 这一栏关注什么

- 面试中高频出现的基础知识点；
- 容易被反复追问的框架和工程问题；
- 复习阶段适合集中整理的知识清单；
- 面试准备过程中的经验和复盘。

## 使用建议

- 如果你正在准备面试，可以把这里作为集中复习入口；
- 如果你平时不面试，也可以把它当成知识点查漏补缺的索引；
- 这部分内容更偏“总结和归纳”，适合搭配基础栏目一起看。
